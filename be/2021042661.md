# 16 JUILLET 2021. - Décret modifiant le Code flamand du Logement de 2021, en ce qui concerne l'ordonnance communale rendant obligatoire l'attestation de conformité

##### Article 1er. Le présent décret règle une matière régionale.

##### Article 2. L'article 3.2, alinéa 1er, 1°, du Code flamand du Logement de 2021 est complété par le membre de phrase " et peut prévoir que, tant que cette obligation n'est pas remplie, une obligation annuelle de demander ce certificat est imposée ".
